⚙️ 설정 안내

1. 메일 설정

email_config.json 파일을 열어 다음을 설정하세요:
- sender_password: 이메일 앱 비밀번호
- 참고: Gmail은 여기에서 앱 비밀번호 생성

2. 자동화 설정 (Windows Task Scheduler)

1) Windows에서 "작업 스케줄러" 검색 및 실행
2) "작업 만들기" 클릭
3) "트리거" 탭에서 매일 10:00에 반복하도록 설정
4) "작업" 탭에서 다음 프로그램 실행:
    프로그램: cmd.exe
    인수 추가: /c "C:\Users\Admin\Desktop\2일차 과제-1\scheduler.bat"

3. 수동 실행 테스트

PowerShell에서 다음 실행:
cd "C:\Users\Admin\Desktop\2일차 과제-1"
python scheduler.bat
